# v7.13.3

# 改进

  1. 独立窗口设置增加标题栏校准。
    Added titlebar height calibration to standalone panel setting.
  2. 设置页面显示版本号。
    Show Saladict verion on options page.

# 修复

  1. 更新 CNKi 接口。
    Updated CNKi.
  2. 修复腾讯翻译君结果输出 HTML 实体。
    Fixed tencent translator showing HTML entities.
  3. 修复麦克米伦词典不能切换美式。
    Fixed Macmillan not able to switch to us version.

# v7.13.2

# 修复

  1. 兼容非英语版本的 Anki。
    Support non-English version of Anki.

# v7.13.0

# 新增

  1. 支持 Anki Connect 自动制卡！见说明。
    Support automatic card creation with Anki Connect! See Notes.
  2. 增加选项记住独立窗口位置与大小。
    Add option for toggling position and size memoization.

# 改进

  1. 查词记录关闭的情况下隐藏查词记录单词本入口。
    Hide Search History page when search history tracking is turned off.
  2. 独立窗口查询剪贴板快捷键在剪贴板为空的情况下依然打开窗口。
    Open standalone panel with "Search clipboard content in Standalone Panel" shortcut even when clipboard is empty.
  3. 关闭窗口的时候同时关闭正在播放的朗读。
    Stop audio playing after panel is closed.
  4. 有道柯林斯支持多入口情况。
    Youdao Collins supports multi-entry.
  5. 查词面板样式细节优化。
    Panel styling improvement.

  • Firefox 商店
  • Edge 商店（由 @rumosky 上传）

墙内及其它国产浏览器可用 Chrome Extension Downloader / Crx4Chrome / 网盘
（⚠️ 请勿解压使用开发者模式安装 ⚠️ 直接拖入 crx 文件到扩展管理页面即可）
（注意谷歌扩展更新了打包格式，部分内核比较滞后的国产浏览器可能安装不了）

国内浏览器自带扩展市场内安装的，对比扩展 id 为 cdonnmffkdaoajfknoeeecmchibpmkmg 即可。

不能享浏览器自动更新的朋友可以关注「柠檬反应堆」公众号或者「沙拉查词」知乎专栏订阅更新动态，我也会在这里分享每次更新背后的想法以及功能设计的理念。
